Tableau 연동 가이드

🔐 인증 방식 선택
Tableau Cloud 임베딩 인증에는 3가지 옵션이 있습니다
Connected Apps (Direct Trust) + JWT 사내 권장
백엔드 서버에서 JWT 토큰을 생성하여 사용자 인증. 별도 IdP 없이 사용 가능. Viewer 라이선스 불필요 (On-Demand Access 활성화 시). 사내 직원 전용 포털에 최적.
1. Tableau Cloud → 설정 → Connected Apps → 새 앱 만들기 2. Client ID와 Secret 복사 3. 백엔드에서 JWT 생성 후 <tableau-viz token="JWT"> 에 전달
Tableau 로그인 세션 기반 (iframe) 가장 간단
직원이 Tableau Cloud에 이미 로그인된 상태에서 iframe으로 직접 임베딩. 추가 설정 불필요. 단, 제3자 쿠키 차단 시 재로그인 필요할 수 있음.
<iframe src="https://YOUR_SITE.online.tableau.com/views/WORKBOOK/SHEET?:embed=y&:toolbar=no" width="100%" height="800"></iframe>
OAuth 2.0 Trust (EAS) 고급
Azure AD, Okta 등 기존 IdP가 있을 때 사용. SSO 연동으로 완전한 무로그인 경험 제공. IdP 설정 필요.
⚡ 빠른 시작: Embedding API v3
1
Tableau Embedding API v3 라이브러리 로드
HTML <head>에 script 태그 추가 (CDN)
<script type="module" src="https://embedding.tableauusercontent.com/tableau.embedding.3.latest.min.js"></script>
2
tableau-viz 웹 컴포넌트 삽입
원하는 위치에 커스텀 HTML 태그 추가
<tableau-viz id="myViz" src="https://YOUR_SITE.online.tableau.com/views/WORKBOOK/SHEET" width="100%" height="800" hide-tabs toolbar="hidden" token="YOUR_JWT_TOKEN"> </tableau-viz>
3
Tableau Cloud에서 대시보드 URL 확인
Tableau Cloud → 대시보드 열기 → 공유 버튼 → 내장 코드 복사 또는 URL 복사
4
Connected Apps 설정 (JWT 인증)
사내 백엔드에서 JWT를 생성하여 token 속성으로 전달. Python 예시:
import jwt, uuid, datetime token = jwt.encode({ "iss": CLIENT_ID, # Connected App Client ID "exp": datetime.datetime.utcnow() + datetime.timedelta(minutes=10), "jti": str(uuid.uuid4()), "aud": "tableau", "sub": user_email, "scp": ["tableau:views:embed"] }, SECRET_VALUE, algorithm="HS256", headers={"kid": SECRET_ID, "iss": CLIENT_ID})
📋 Viewer 라이선스 없이 임베딩하는 방법 (On-Demand Access)
Tableau Cloud의 On-Demand Access (ODA) 기능을 활성화하면, Tableau 라이선스가 없는 사내 직원도 JWT를 통해 임시 세션으로 대시보드를 열람할 수 있습니다.

설정 경로: Tableau Cloud → 설정 → Connected Apps → 앱 편집 → On-Demand Access 활성화
# JWT에 ODA 관련 클레임 추가 { "iss": CLIENT_ID, "sub": "viewer-user@company.com", # 임의 이메일 가능 (ODA 활성화 시) "aud": "tableau", "scp": ["tableau:views:embed"], "exp": ..., "jti": ... }
