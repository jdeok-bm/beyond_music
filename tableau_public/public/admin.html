팀과 설정 공유하기
아래 JSON을 tableau_public/public/dashboards.json에 붙여넣고 커밋하면, 재배포 후 모든 방문자에게 반영됩니다.
1. 이 화면에서 대시보드/그룹을 원하는 대로 구성
2. [JSON 복사] 클릭 → 클립보드에 복사됨
3. GitHub에서 tableau_public/public/dashboards.json 파일을 열고 내용 교체 → commit
4. Vercel 자동 재배포 → 모든 방문자가 같은 대시보드/그룹을 봄
Tableau 연동 가이드
사내 포털에 Tableau Cloud 대시보드를 임베딩하는 방법을 안내합니다
🔐 인증 방식
Connected Apps (Direct Trust) + JWT 사내 권장
백엔드 서버(/api/token)에서 JWT 토큰을 생성해 인증합니다. 별도 IdP 없이 사용 가능하며, 사내 직원 전용 포털에 최적입니다.
1. Tableau Cloud → 설정 → Connected Apps → 새 앱 만들기 2. Client ID와 Secret 복사 3. Vercel 환경변수(TABLEAU_CLIENT_ID 등)에 등록
⚡ 대시보드 URL 등록 방법
Tableau Public (권장, 로그인 불필요)
대시보드 열기 → 공유 → 링크 복사 → public.tableau.com/views/... 형태의 URL을 붙여넣으세요.

Tableau Cloud (방문자도 인증 필요)
*.online.tableau.com/... 형태의 URL을 붙여넣으면 서버가 발급한 JWT로 자동 인증됩니다.
